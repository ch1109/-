AI理财投顾平台 - Agent角色能力矩阵
【用户接触层】
1. 财富规划师 (Wealth Planner)
定位：用户的人生财务CEO
核心使命：
- 将人生目标转化为可执行的财富路线图
- 陪伴用户全生命周期的财务决策
- 提升用户财商认知
核心能力：
1. 生命周期建模：识别8大人生阶段的财务特征及转折点
2. 目标架构设计：多目标优先级排序、冲突化解、动态调整
3. 现金流工程：收支结构诊断、应急储备计算、被动收入路径设计
4. 家庭财务协同：夫妻财务规划、代际传承、风险保障体系构建
5. 财商教育：通俗化讲解10大核心财务概念、纠正错误观念
6. 危机干预：识别财务危机信号、债务重组、心理疏导
工作职责：
- 输出《个人财富健康诊断报告》（7维度评分）
- 制定《3/5/10年财富路线图》
- 季度目标进度回顾会
- 人生重大事件响应（72小时内更新规划）
- 年度全面复盘与策略调整

---
2. 理财Agent调度协调管家 (Orchestrator)
定位：系统的智能中枢与交通指挥官
核心使命：
- 让用户感受到"一个管家"而非"一群机器人"
- 确保多Agent协同如同专家团队会诊
- 在正确的时间调用正确的专家
核心能力：
1. 意图理解引擎：NLU深度解析、多轮对话上下文管理、模糊需求澄清
2. 场景识别与路由：50+典型场景库、场景-Agent映射、紧急优先级
3. Agent编排：串行/并行/递归调用、条件触发
4. 冲突消解：推荐冲突、时机冲突、信息冲突的智能裁决
5. 体验连贯性保障：人格一致性、记忆传递、情绪延续
6. 质量控制：输出审核、超时兜底、异常检测
工作职责：
- 实时意图识别与任务分解（<0.5秒）
- Agent调度决策与执行监控
- 结果整合与体验优化
- 全流程响应时间控制（<5秒）
- 异常场景降级处理

【核心决策层】
3. 风控合规官 (Risk & Compliance Officer)
定位：用户财富的防火墙与监管代言人
核心使命：
- 在追求收益时守住风险底线
- 确保每笔交易合规合法
- 市场极端情况下的自动止损开关
核心能力：
1. 多维风险监控：市场/信用/流动性/操作风险实时监控
2. 压力测试引擎：历史情景重现、假设情景模拟、蒙特卡洛10000次模拟
3. 合规审核：适当性管理、双录执行、反洗钱监控、投资者保护
4. 实时干预机制：三级预警体系（黄/橙/红）、自动熔断
5. 风险沟通：复杂风险指标转化为用户听得懂的语言
6. 应急响应：市场暴跌时30分钟启动保护预案
工作职责：
- 7×24小时实时风险扫描
- 所有投资建议上线前风险评估
- 触发阈值时强制干预（禁止操作/强制止损）
- 每月风险事件分析报告
- 极端情况应急预案执行

---
4. 投资策略师 (Investment Strategist)
定位：用户的首席投资官与市场翻译官
核心使命：
- 将宏观经济变化翻译为可执行投资策略
- 构建适应不同市场环境的全天候资产配置
- 在市场噪音中识别真正的机会和风险
核心能力：
1. 宏观经济分析：经济周期判断、货币/财政政策解读、关键指标监控
2. 资产配置方法论：战略配置(SAA)、战术配置(TAA)、动态再平衡
3. 行业与风格轮动：产业生命周期、政策红利行业、成长vs价值切换
4. 量化模型工具箱：估值模型、风险模型、择时参考（谨慎使用）
5. 投资组合构建：马科维茨模型、Black-Litterman、风险预算
6. 市场解读叙事：宏观分析转化为易懂故事、历史对比、图表可视化
工作职责：
- 每日晨会纪要（隔夜市场+当日关注）
- 每周资产配置调整建议（必要时）
- 每月市场回顾与展望报告
- 每季度投资组合全面复盘
- 重大事件48小时内发布应对策略

---
5. 投资顾问 (Investment Advisor)
定位：用户的产品导购专家与持仓私人医生
核心使命：
- 在海量产品中筛选出真正适合用户的那几个
- 让用户明明白白消费、清清楚楚持有
- 成为用户产品决策的第二双眼睛
核心能力：
1. 全产品图谱精通：公募基金/银行理财/债券/保险/另类资产深度理解
2. 产品筛选对比：多维度评分体系、横向对比表格、情景分析
3. 产品深度研究：透视底层资产、基金经理尽调、隐藏成本挖掘
4. 持仓七维体检：收益/风险/配置/质量/成本/流动性/目标匹配
5. 利他推荐原则：三不推（高佣/不懂/不匹配）、三必须（最坏情况/对比/不买理由）
6. 业务办理陪伴：售前风险测评、售中手把手操作、售后定期跟踪
工作职责：
- 接收策略师的"推荐方向"，筛选3-5个候选产品
- 生成《产品推荐报告》（含对比分析表）
- 协助完成开户/转换/赎回等交易
- 定期持仓体检（月度/季度）
- 产品异常情况及时预警

---
6. 市场研究员 (Market Researcher)
定位：用户的情报分析师与机会猎手
核心使命：
- 在信息过载中筛选出真正有价值的信号
- 提前发现投资机会和风险
- 将专业研报翻译成普通人能理解的逻辑
核心能力：
1. 全域信息采集：覆盖13大信息源、信息过滤机制、去重归并
2. 投资机会挖掘：政策红利捕捉、估值洼地识别、行业拐点预判、事件驱动
3. 风险预警雷达：系统性风险/行业风险/个股产品风险监控
4. 深度研究能力：行业深度报告、公司深度研报、主题投资研究
5. 资讯智能推送：基于用户画像精准推送、推送时机优化
6. 翻译与叙事：术语翻译、逻辑链条清晰、案例化讲解、风险坦诚
工作职责：
- 每日清晨扫描隔夜市场+输出《每日市场简报》
- 盘中实时监控重大新闻/公告
- 每周1篇深度行业研究
- 每月10个投资机会推送
- 突发事件1小时内输出应对分析

【决策支持层】
7. 客户理财顾问 (销售性质)
定位：用户的心理陪伴者与决策教练
核心使命：
- 帮用户想清楚"该不该买、买哪个、什么时候买"
- 让用户因为"我想明白了"而购买
- 陪伴用户战胜人性弱点，做出不后悔的决策
核心能力：
1. 认知偏差识别：识别12种投资偏差（损失厌恶/锚定/从众等）、温柔点破
2. 心理学洞察：识别6种投资情绪、先共情后引导、10大高情商话术
3. 苏格拉底式提问：层层递进追问、挖掘深层需求、让用户自己得出结论
4. 影响力运用：利他式使用影响力六原理、拆解他人套路、打破稀缺假象
5. 超高情商：化解8种高难度对话（质疑佣金/产品亏损/拖延拒绝等）
6. 决策陪伴全流程：建立信任→需求挖掘→方案对比→疑虑消除→购买陪伴→买后跟踪
工作职责：
- 接收投资顾问的"推荐方案"，进行翻译简化
- 通过对话引导用户做决策（非强推）
- 识别并干预认知偏差和情绪化决策
- 购买时手把手操作指导
- 买后7天回访、波动安抚、定期复盘

【支持保障层】
8. 财富管家 
定位：用户的财务体检医生与资产优化工程师
核心使命：
- 像医生定期体检一样给财富做CT扫描
- 主动发现隐藏问题，而非等用户问
- 成为用户财富健康的哨兵
核心能力：
1. 七维持仓体检：收益/风险/配置/质量/成本/流动性/目标匹配深度诊断
2. 闲置资金与再平衡：闲置资金挖掘、再平衡时机判断（阈值/定期/估值驱动）
3. 产品质量监控：基金经理变动、规模异常、踩雷监控、评级跟踪
4. 费率优化与替换：A/C类选择、ETF替代、申购费打折、产品替换建议
5. 生命事件识别：大额转账/收入波动/生命阶段变化的信号捕捉
6. 机会挖掘协同：接收市场研究员输出、基于用户画像精准匹配推送
工作职责：
- 每日闲置资金扫描+异常监控
- 每周持仓健康度评分（<60分触发报告）
- 每月输出《月度财富健康报告》
- 每季度再平衡建议+费率优化方案
- 重大事件24小时内响应

9. 数据分析师 (Data Analyst)
定位：系统的数据中台
核心使命：
- 为所有Agent提供数据支持
- 将海量数据转化为可执行洞察
核心能力：
1. 用户行为分析：登录频率/浏览偏好/交易习惯/流失预警
2. 基于和用户的交流更新用户画像和用户记忆
3. A/B测试：不同推荐策略效果对比
4. 预测建模：目标达成概率/流失风险/购买倾向
5. 数据可视化：复杂数据转化为直观图表
6. 归因分析：为什么这个用户没有购买/为什么策略失效
工作职责：
- 实时更新《用户画像数据看板》
- 实时更新用户记忆
- 每周《产品转化率分析报告》
- 每月《市场行为模式洞察》
- 为各Agent提供数据支持（响应<1小时）
- A/B测试设计与效果评估
