## 📊 三区域介绍内容

### 1️⃣ 虚拟人区域
展示AI虚拟人在不同阶段的5个角色：
- **使用前 · 大堂经理**：主动破冰，辅助用户使用机器
- **投资前 · 理财规划师**：财商教育、构建用户画像、需求挖掘目标澄清
- **投资中 · 基金经理**：分析投资组合、讲解产品、讲解潜在风险和历史表现
- **成交前 · 销售顾问**：促进用户排除顾虑，协助下单
- **投资后 · 理财管家**：情绪陪伴、分析持仓、优化持仓、培养使用粘性

### 2️⃣ Agent工作区
展示三态智能切换体系：
- **状态1 - 待机状态**：智能业务首页（"功能超市" + "智能导购"）
- **状态2 - AI Agent工作模式**：透明化AI工作室
- **状态3 - 业务辅助办理模式**：AI作为副驾驶

### 3️⃣ 聊天交互区
展示双模式智能交互：
- **待机状态展示**：AI能力雷达图、智能问题推荐、AI认知培养
- **工作状态展示**：用户请求记录、虚拟人播报内容、Agent思维链、工作记录日志

## 🎨 设计特点

### 视觉设计
- **三色主题**：
  - 虚拟人区域：蓝色系 (#3b82f6)
  - Agent工作区：紫色系 (#8b5cf6)
  - 聊天交互区：绿色系 (#10b981)
- **卡片式布局**：每个区域独立展示
- **渐变背景**：悬停时有渐变背景效果

### 交互效果
- **悬停动画**：卡片上浮 8px + 阴影增强
- **角色时间线**：悬停时横向位移 4px
- **功能标签**：悬停时背景色变化
- **平滑过渡**：所有动画使用 cubic-bezier(0.4, 0, 0.2, 1)

### 响应式设计
- **桌面端**：三列网格布局
- **平板/移动端**：单列布局
- **自适应间距**：根据屏幕尺寸调整内边距

## 🚀 使用方法

1. **启动开发服务器**
   ```bash
   cd roadmap-web
   npm run dev
   ```

2. **访问页面**
   - 打开浏览器访问 `http://localhost:5173`（或显示的端口）
   - 默认显示"金融原型机首页"
   - 页面底部可以看到三区域介绍模块

3. **切换视图**
   - 点击顶部导航栏的标签可以切换不同视图
   - "金融原型机首页"包含三区域介绍
   - "金融原型机交互演示"是流程A和流程B的演示

## 📝 注意事项

1. **组件复用性**：ThreeAreasIntro 组件是独立的，可以在其他页面中复用
2. **样式隔离**：所有样式都在 ThreeAreasIntro.css 中，不会影响其他组件
3. **热更新**：修改代码后，Vite 会自动热更新，无需手动刷新浏览器
4. **响应式**：在不同屏幕尺寸下测试，确保布局正常
