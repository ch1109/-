### 1. 移除了Agent工作区的时间标记和步骤标题

**修改位置**: `renderScenarioWorkspace` 函数

**之前**:
```tsx
<header className="scenario-header">
  <div className="scenario-heading">
    <span className="scenario-time">{step.time}</span>  {/* T=0s */}
    <div className="scenario-headline">
      <span className="scenario-badge">{badge}</span>  {/* 待机态首页 */}
      <h4>{formatWithContext(step.title, context)}</h4>
    </div>
  </div>
  {workspace.progress && (
    <span className="scenario-progress">{formatWithContext(workspace.progress, context)}</span>
  )}
</header>
<div className="scenario-body">
  <h5>{headline}</h5>
  {hasDescription && <p>{description}</p>}
</div>
```

**现在**:
```tsx
{/* 移除了时间标记、badge和步骤标题 */}
{workspace.progress && (
  <div className="scenario-progress-indicator">
    <span className="scenario-progress">{formatWithContext(workspace.progress, context)}</span>
  </div>
)}
```

**效果**: Agent工作区不再显示"T=0s · 待机态首页"等解释性文字,只显示卡片、数据、图表等AI生成的结果
